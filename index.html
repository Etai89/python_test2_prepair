דף הכנה למבחן 2 - מבני נתונים ולולאות בפייתון

רשימות (Lists) - מבנה נתונים חיוני

רשימה היא מבנה נתונים שיכול להכיל נתונים מאותו הסוג או מסוגים שונים - זו התשובה הנכונה!

⚠️ הערה חשובה:
  • רשימה בפייתון יכולה להכיל כל סוג נתונים ביחד
  • האיבר הראשון ברשימה נמצא במיקום 0 - לא 1!
  • זה נקרא "אינדקס" והוא מתחיל תמיד מ-0
  • ניתן לגשת לאיברים עם סוגריים מרובעים []
  • הדוגמאות בדף דומות למבחן אבל לא זהות - כדי שתבינו את העקרונות
יצירת רשימות ואינדקסים:
# יצירת רשימה my_list = [10, 20, 30, 40] # גישה לאיברים - אינדקס מתחיל מ-0! print(my_list[0]) # יודפס: 10 (איבר ראשון) print(my_list[1]) # יודפס: 20 (איבר שני) print(my_list[3]) # יודפס: 40 (איבר רביעי) # רשימה מעורבת mixed_list = ["שם", 25, True, 3.14]

לולאת For עם רשימות - מושג מרכזי במבחן

דוגמה טיפוסית: מה התוכנית תדפיס?
for i in [1, 2, 3]: print(i * 3)
התשובה: 3, 6, 9
לולאה על רשימה
for i in [1, 2, 3]: print(i * 3)

תדפיס: 3, 6, 9

איך זה עובד?

i=1 → 1*3 = 3
i=2 → 2*3 = 6
i=3 → 3*3 = 9

לולאה על מחרוזות
for name in ["דני", "שרה"]: print("שלום", name)

פונקציית Range - הבנה מעמיקה

מה קובע ה-range בכתיב הזה? for i in range(n)
תשובה נכונה: קובע את תחום המספרים כמה פעמים יש לבצע את גוף הלולאה

שלוש צורות של Range - חובה לדעת!

range(n)
for i in range(5): print(i) # יודפס: 0, 1, 2, 3, 4

מ-0 עד n-1

range(start, stop)
range(3, 8) # מייצר: 3, 4, 5, 6, 7

מ-start עד stop-1

range(start, stop, step)
range(1, 10, 3) # מייצר: 1, 4, 7

עם צעד של 3

⚠️ שים לב למלכודות נפוצות:
print(range(2, 7, 1)) - זה לא יודפס כרשימה!
תצטרך לכתוב: print(list(range(2, 7, 1)))
התוצאה הנכונה: [2, 3, 4, 5, 6]
דוגמאות טיפוסיות:
# מה יודפס? (דוגמה טיפוסית) print(list(range(2, 7, 1))) # [2, 3, 4, 5, 6] print(list(range(1, 10, 3))) # [1, 4, 7] # לולאה עם range for i in range(1, 6, 2): print(i) # יודפס: 1, 3, 5

מילונים (Dictionaries) - מושג חיוני

מה זה Dictionary?
התשובה הנכונה: טיפוס שבאמצעותו משייכים מפתח לערך
(לא יומן ולא מערך!)
יצירת מילון
student = { "name": "דני", "age": 20, "grade": 85 }
גישה לערכים
# דרך ישירה print(student["name"]) # דרך בטוחה עם get() print(student.get("name"))
בדיקת קיום מפתח
if "name" in student: print("מפתח קיים")

פונקציות חשובות במילון - מה חובה לדעת למבחן

get(key) - מה הפונקציה עושה?
תשובה נכונה: מחזירה את הערך שתואם למפתח

מה קורה אם המפתח לא קיים?
תשובה נכונה: יחזור None (לא 0 ולא 1!)
דוגמאות נפוצות:
info_dict = {"x": 5, "y": 10} # get() עם מפתח קיים print(info_dict.get("x")) # יודפס: 5 # get() עם מפתח לא קיים print(info_dict.get("z")) # יודפס: None # בדיקת קיום מפתח עם in if "y" in info_dict: print("המפתח y קיים!")
מה הפונקציה in עושה?
התשובה הנכונה: בודקת האם מפתח כלשהו נמצא במילון
(לא מכניסה מפתח או ערך!)

Tuples (רשימות קבועות)

באיזה סוגריים נאתחל Tuple?
התשובה הנכונה: () סוגריים עגולות
(לא { } ויש דבר כזה Tuple!)
יצירת Tuple
# Tuple עם מספר איברים point = (3, 5) # Tuple עם איבר אחד - שים לב לפסיק! single = (42,)
גישה לאיברים
colors = ("red", "green", "blue") print(colors[0]) # red print(colors[2]) # blue
ההבדל מרשימה

Tuple: (1, 2, 3) - קבוע
List: [1, 2, 3] - ניתן לשינוי

💡 דברים שאסור לשכוח במבחן!

🚨 נקודות קריטיות שקובעות הצלחה או כישלון:

  • אינדקס ראשון תמיד 0! - ברשימות, Tuples ומחרוזות
  • get() מחזיר None - לא שגיאה ולא 0
  • in רק בודק - לא מכניס ערכים למילון
  • range(n) לא כולל את n - רק 0 עד n-1
  • Tuple עם סוגריים עגולות () - לא [] או {}

📐 הנוסחאות החשובות

📋 רשימות
# יצירה my_list = [1, 2, 3] # גישה (זכרו: מתחיל מ-0!) first = my_list[0] # האיבר הראשון
זכרו: מבנה נתונים שיכול להכיל כל דבר
🔄 לולאות עם Range
# מ-0 עד n-1 for i in range(5): print(i) # 0,1,2,3,4 # מ-start עד stop-1 for i in range(2, 5): print(i) # 2,3,4
זכרו: קובע כמה פעמים הלולאה רצה
📖 מילונים
# יצירה data = {"key": "value"} # גישה בטוחה result = data.get("key") # בדיקת קיום if "key" in data:
זכרו: משייך מפתח לערך
📦 Tuples
# יצירה (שימו לב לסוגריים!) point = (3, 5) # עם איבר אחד - חובה פסיק! single = (42,)
זכרו: כמו רשימה אבל קבועה

🎯 מלכודות נפוצות - תזהרו!

❌ שגיאות שעושים הכי הרבה:

# ❌ חושבים שהאיבר הראשון הוא 1 numbers = [10, 20, 30] print(numbers[1]) # זה ידפיס 20, לא 10! # ✅ נכון - האיבר הראשון תמיד באינדקס 0 print(numbers[0]) # זה ידפיס 10 # ❌ חושבים ש-get() יגרום לשגיאה info = {"age": 25} print(info.get("name")) # לא שגיאה! יודפס None # ❌ מבלבלים בין סוגריים wrong_tuple = [1, 2] # זו רשימה! correct_tuple = (1, 2) # זה Tuple!

🧠 איך לזכור - טריקים מנטליים

💭 דרכים לזכור את הדברים החשובים:

  • אינדקס 0: "מתכנתים סופרים מ-0, לא מ-1!"
  • range(5): "5 מספרים: 0,1,2,3,4 (בלי ה-5!)"
  • get() בטוח: "get לא נותן שגיאה, רק None"
  • in רק בודק: "in זה כמו 'האם יש?' - לא מוסיף כלום"
  • Tuple עם (): "Tuple = רשימה עם סוגריים עגולות"

📊 טבלת השוואה מהירה

מבנה נתונים סוגריים ניתן לשינוי? אינדקס ראשון דוגמה
List (רשימה) [ ] כן 0 [1, 2, 3]
Tuple ( ) לא 0 (1, 2, 3)
Dictionary (מילון) { } כן מפתח {"a": 1, "b": 2}

שגיאות נפוצות - חשוב לא לפספס!

השגיאות שגורמות לכישלון במבחן:
  • לחשוב שאינדקס ראשון הוא 1 - הוא תמיד 0!
  • לבלבל בין range צורות שונות - תרגלו היטב
  • לחשוב ש-get() זורק שגיאה - הוא מחזיר None
  • לבלבל בין List ל-Tuple - סוגריים שונים!
  • לחשוב ש-in מכניס ערך - הוא רק בודק קיום
  • לשכוח פסיק ב-Tuple עם איבר אחד - (42,) לא (42)

דוגמאות קוד מקיפות

עבודה עם רשימות ולולאות

# יצירת רשימה ועבודה איתה grades = [85, 90, 78, 92] # הדפסת כל הציונים for grade in grades: print("ציון:", grade) # גישה לציון הראשון print("ציון ראשון:", grades[0])

עבודה עם מילונים

# יצירת מילון סטודנט student = { "name": "שרה", "age": 22, "grade": 95 } # בדיקה אם מפתח קיים if "name" in student: print("שם הסטודנט:", student.get("name")) # שליפת ערך שלא קיים print(student.get("phone")) # יחזיר None

עבודה עם Tuples

# יצירת Tuple עם קואורדינטות location = (32.0853, 34.7818) # תל אביב # שליפת קואורדינטות lat = location[0] lng = location[1] print("קו רוחב:", lat) print("קו אורך:", lng)

טיפים להכנה יעילה למבחן

אסטרטגיית הכנה

  • תרגלו את כל שלוש צורות ה-range - הן יופיעו במבחן!
  • זכרו: אינדקס ראשון תמיד 0 - תחזרו על זה עד שיישאר
  • הבינו את ההבדל בין get() לגישה ישירה - חשוב מאוד
  • תרגלו סוגריים: [ ] לרשימה, ( ) ל-Tuple, { } למילון
  • תבינו מה in עושה - רק בודק קיום, לא מכניס
ביום המבחן - רשימת בדיקה:
  • ✅ זכרתי שאינדקס ראשון הוא 0?
  • ✅ הבנתי איך range עובד?
  • ✅ יודע מה get() מחזיר כשמפתח לא קיים?
  • ✅ זוכר באיזה סוגריים יוצרים Tuple?
  • ✅ יודע מה in בודק במילון?

🎯 מה כדאי לעשות בשעות האחרונות לפני המבחן?

💪 חזרה אחרונה - בדקו שאתם יודעים:

  1. לכתוב לולאה עם range:
    for i in range(10): # מדפיס 0-9
  2. לגשת לאיבר ברשימה:
    my_list[0] # איבר ראשון!
  3. לבדוק מפתח במילון:
    if "key" in my_dict:
  4. להשתמש ב-get בביטחון:
    value = my_dict.get("key") # None אם אין
  5. ליצור Tuple:
    point = (3, 5) # סוגריים עגולות!

🚨 הדברים שיכולים לקלקל לכם את הציון:

  • לכתוב numbers[1] במקום numbers[0] לאיבר הראשון
  • לחשוב שrange(5) כולל את 5 - זה 0,1,2,3,4 בלבד!
  • לכתוב רשימה במקום Tuple - [1,2] זו רשימה, (1,2) זה Tuple!
  • לחשוב שget() זורק שגיאה - זה מחזיר None בשקט

אם אתם לא בטוחים, בדקו עם דוגמה קטנה במחשבה!

💪 תרגילים לתרגול

תרגיל 1: עבודה עם רשימות ואינדקסים

שאלה: מה יודפס בקוד הבא?
numbers = [15, 25, 35, 45, 55] print(numbers[0]) print(numbers[3])
👁️ לחץ לפתרון
תשובה:
15 (האיבר הראשון באינדקס 0)
45 (האיבר הרביעי באינדקס 3)

תרגיל 2: לולאות עם range

שאלה: מה יודפס בקוד הבא?
for i in range(2, 8, 2): print(i)
👁️ לחץ לפתרון
תשובה:
2
4
6
הסבר: מתחיל מ-2, מסיים לפני 8, קופץ ב-2 כל פעם

תרגיל 3: מילונים ו-get()

שאלה: מה יודפס בקוד הבא?
person = {"name": "דוד", "age": 30} print(person.get("name")) print(person.get("phone")) print(person.get("age"))
👁️ לחץ לפתרון
תשובה:
דוד (המפתח "name" קיים)
None (המפתח "phone" לא קיים - get מחזיר None)
30 (המפתח "age" קיים)

תרגיל 4: בדיקת מפתח במילון

שאלה: מה יודפס בקוד הבא?
data = {"x": 10, "y": 20} if "x" in data: print("קיים") if "z" in data: print("נמצא") else: print("לא נמצא")
👁️ לחץ לפתרון
תשובה:
קיים (כי "x" נמצא במילון)
לא נמצא (כי "z" לא נמצא במילון)

תרגיל 5: Tuples וגישה לאיברים

שאלה: מה יודפס בקוד הבא?
colors = ("אדום", "ירוק", "כחול") print(colors[0]) print(colors[2])
👁️ לחץ לפתרון
תשובה:
אדום (איבר ראשון באינדקס 0)
כחול (איבר שלישי באינדקס 2)

תרגיל 6: לולאה על רשימה

שאלה: מה יודפס בקוד הבא?
fruits = ["תפוח", "בננה", "תפוז"] for fruit in fruits: print("אני אוהב", fruit)
👁️ לחץ לפתרון
תשובה:
אני אוהב תפוח
אני אוהב בננה
אני אוהב תפוז

תרגיל 7: range עם הדפסה

שאלה: מה יודפס בקוד הבא?
print(list(range(3, 10, 2)))
👁️ לחץ לפתרון
תשובה:
[3, 5, 7, 9]
הסבר: מתחיל מ-3, מסיים לפני 10, קופץ ב-2 כל פעם

תרגיל 8: לולאה עם חישובים

שאלה: מה יודפס בקוד הבא?
for num in [2, 4, 6]: print(num * 5)
👁️ לחץ לפתרון
תשובה:
10 (2 * 5)
20 (4 * 5)
30 (6 * 5)

תרגיל 9: מילון ולולאה

שאלה: מה יודפס בקוד הבא?
grades = {"math": 90, "english": 85} for subject in grades: print(subject)
👁️ לחץ לפתרון
תשובה:
math
english
הסבר: לולאה על מילון מחזירה את המפתחות (לא הערכים!)

תרגיל 10: שאלה מאתגרת

שאלה: מה יודפס בקוד הבא?
items = [10, 20, 30] for i in range(len(items)): print(items[i])
👁️ לחץ לפתרון
תשובה:
10
20
30
הסבר: len(items) = 3, אז range(3) מייצר 0,1,2
items[0]=10, items[1]=20, items[2]=30
